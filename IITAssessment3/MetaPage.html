Adherence to design specifications

  1. The topic is "Why We Love Vintage?" found on the canvas list. This title was somewhat ambiguous, so I chose to write about the validity of using retro technolody in the modern day.

  3. Images are included as part of the sidebar on every page , as well as several illustrative images used throughout the content.
  4. Hyperlinks are used for both navigation within the site, and for external links. The sidebar features hyperlinks to other pages , while external links are used in the main content, as well as on the references page in place of the URLs.
  5. A contact form has been added at Contact.html
  6. Lists are present on both this page (ordered list for the design specifications), and on the sidebar across all pages (unordered list for the links to other pages.)
  7. All pages use UTF-8 characters, as seen usually on line 6 of every page.
  8. hrs are used on the main content page, along side elements such as i for italics and strong for the quote.
  9. div, header, nav, and main elemetns are present on all content pages, with the appropriate sections contained within them
  10. CSS styling is done externally in ExternalStyle.css, there also the other one
  11. Css properties have been done, I just need to document where
  12. All colors are in a hexcode format, as seen throughout the style sheet.
